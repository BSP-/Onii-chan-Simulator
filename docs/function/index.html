    public split(ctx: object, string: string, fontSize: number, maxWidth: number, doGuess: bool, output: string[]): string[] source 

    import {split} from 'eVN/src/Visuals/text.js'

    Explodes string into an array of strings that won't exceed maxWidth when drawn on ctx

    Params:

    NameTypeAttributeDescription
    ctx object

    Context to probe .measureText() on
    string string

    The string to split
    fontSize number

    Font size of the context
    maxWidth number

    The maximum width of a string in the returned array (in pixels)
    doGuess bool
    • optional
    • default: true

    Try estimating the line width using the font size and maxWidth propert
    output string[]
    • optional

    An argument used by the function when recursing when concatenating previous attempts

    Return:

    string[]

    An array containing slices of string with a maximum pixel width of maxWidth
